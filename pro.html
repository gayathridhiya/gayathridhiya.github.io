PROJECTS

SECURE MODEL FOR DATA ON CLOUD USING EDGE HYBRID ENCRYPTION AND STEGANOGRAPHY

Secure Model for Data on Cloud using Edge Hybrid Encryption and Steganography is a software project which provides the flexibility of passing information by implementing the Steganography & Hybrid Cryptography encryption techniques through a user friendly Graphical user interface in such a way that reduces the manual overhead of verification and paper work of sensitive documents possessed by an individual. Sensitive documents are kept without the harm of being tampered.

SPORT TIPSTER

The Sport Tipster is a database decision support system which aids in successful profit making betting services. This database maintains a detailed record of the horse racing environment which helps in placing sophisticated bets. The user can use this database to identify the horses that have a high chance of winning and to analyze the trends in racing environment.

SITE SCHEDULER

Site Scheduler is a python package which allows scheduling of online sites or online courses. This project significantly replaces bookmarks for remembering sites and also invokes the specific website automatically when the scheduled time is met. It greatly aids students and professionals in organizing their online schedules and sticking to time disciplines.

SMART SURVEILLANCE

Smart Surveillance is a monitoring system which automatically initiates recording and alerts the owner in case of intrusions in the surveillance area. It allows only the concerned person to view the details. Necessary actions can be taken in a short span of time in case of emergency situations.
